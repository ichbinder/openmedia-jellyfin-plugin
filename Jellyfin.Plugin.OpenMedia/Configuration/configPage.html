openmedia

Verbindet Jellyfin mit deinem openmedia-Account. Filme deiner UserLibrary erscheinen als virtuelle Library und werden direkt aus S3 gestreamt.

Basis-URL der openmedia-API. Standard: https://api.mediatoken.de.
Persönlicher API-Token aus dem openmedia-Profil unter API-Tokens. Beginnt mit om_.
Absoluter Pfad zum STRM-Verzeichnis. Leer lassen fuer den Default unter <data>/openmedia-strm. Das Plugin legt das Verzeichnis beim Start automatisch an.
Name der Jellyfin-Library die das Plugin automatisch anlegt. Standard: openmedia.
Wenn aktiv, prueft das Plugin beim Jellyfin-Start ob bereits eine Library mit dem oben gesetzten Namen existiert. Falls nicht, wird sie als Movies-Library mit dem STRM-Verzeichnis als Quelle angelegt. Existierende Libraries werden nie veraendert.
